Automatic V2Ray VPN Server Installation on VPS

Set up your secure V2Ray VPN server in minutes with this fully automated installation script. Works seamlessly with WebSocket over port 443 and supports major Linux distributions that use APT and systemd.

Key Features

  • Uses TLS (Transport Layer Security), secure and stealthy against deep packet inspection (DPI)
  • Create and manage multiple VPN profiles
  • Profiles saved in v2ray-profiles folder
  • Revoke profiles and remove their configurations
  • Completely uninstall V2Ray and clean up the system

Supported Operating Systems

The script is designed for Linux distributions using apt-get and systemctl:

  • Ubuntu (16.04, 18.04, 20.04, 22.04 and newer)
  • Debian (9 "Stretch", 10 "Buster", 11 "Bullseye" and newer)
  • Linux Mint (Ubuntu-based)
  • Raspbian / Raspberry Pi OS (with systemd)
  • Elementary OS (Ubuntu-based)
  • Pop!_OS (Ubuntu-based)
  • Zorin OS (Ubuntu-based)

Recommended VPN Client

You can use V2Box as your VPN client for the best compatibility and performance.

Installation Instructions

Run the following command with root privileges on a clean VPS:

wget -qO- https://v2ray-install.pages.dev/v2ray.sh | bash

The setup uses WebSocket on port 443. For optimal performance, use a fresh VPS not running other web services.

Important: Ensure that TCP port 443 is open for incoming connections in your VPS firewall settings.
